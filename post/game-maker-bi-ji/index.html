2020-11-17

Project1: BreakThrough DnD

名词解释

sprites: 表示游戏世界中物体的像素点集合
objects: 程序处理的对象，是像素点集合在程序中的指称
rooms: 游戏世界中的空间、数值变化的区间、sprites移动的边界
events: 游戏中发生的事件，大体可分成生、变、灭、入
actions: 对events的具体表述（or结果），例如何时生、何时变、如何生、如何变
events和actions共同决定了objects对什么事件做出何种反应。

系统分析

世界观：Ball、Bat、Brick的设定和活动的空间
碰撞系统：三个物体之间的关系，何为碰撞、如何反弹（向量）
状态系统：玩家分数、最高分数和生命值

容易混淆的点

rooms中的物件并不是objects本身，只是objects的copy。这意味着这些copies在游戏中可以有不同的表现形式（通过objects的属性实现），但不改变objects本身。但是一旦改变objects本身，所有copies都会随之改变。好优美的逻辑关系有没有

  2. 得到扩大buff后，再得到加速buff会跳出错误
