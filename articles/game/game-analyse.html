游戏体验与分析

映月城与电子姬体验

新手引导

  1. 战斗系统（闪避+反闪攻击+属性克制）
  2. 武器系统
  3. 卡牌系统
  4. 人物强化系统
  5. 潘多拉系统
  6. 蓝图系统*1/2

有趣的点

  1. 制作人的惊悚悬疑桥段，潘多拉系统→直面国内手游市场的规则
  2. 请假系统→让人耳目一新，也有卖情怀的影子，具体能否增加用户粘性仍待考证
  3. 制作人在游戏内的频繁出现→虚拟和现实交织的感觉，最强蜗牛亦如是

不好的点

  1. 资源消耗界面没有直观显示资源现有量
  2. 通过邮件的方式展示剧情or设置悬疑真的很难让人快速get到点，碎片化的消息处理起来很恶心

初步评分

世界观与剧情：⭐⭐
常见的未来题材，故事主要围绕被病毒感染的仿生人展开，暂时来看没有让我感到印象深刻。
立绘：⭐⭐
怎么说的，感觉到有在尽力模仿那个游戏，但那是人家的看家本领啊喂……对比下来你这儿的立绘就好丑啊。
战斗：⭐⭐⭐
卡通人物2D射击令人耳目一新，既涉及射击的主题，又比刺激战场等更加轻量化，以及降低了操作性，值得一个好评，总之选择这个玩法作为突围点是很ok的;抄过来一个编队，但是人物不能同时作战，葫芦娃救爷爷也太惨了吧哈哈哈，编队的意义何在;策略性低、操作空间小，再强的boss靠走位+反击就能干掉，中后期可能会无法给玩家提供成就感;
UI：⭐
理由：我相信以其他界面的设计水准，充值界面不至于搞成这样，那么故意做成页游风格是在恶心谁？

收获

1.玩家角色、引导者和卡牌的身份代入问题是世界观的重要入口，要做好包装
2.具体到我们现在的游戏，就是必须明确玩家是以何种视角进入到世界中，小精灵的角色从何而来以及卡牌的包装。
3.游戏第一/三人称视角的利弊分析：
  • 第一人称
    • 1.能够埋一根长线悬疑（失忆→我是谁？世界为何？）伴着主线的进展，“我”的身份逐渐揭开，世界也自然而然在“我”前被逐渐发现，这其实是一个recover的过程，能为剧情的大转留下不错的空间。
    2.代入感（第一人称就更有代入感吗？个人感觉真的未必从台词选择的角度来看确实有效果，但另一方面来看手游的声系统营造的代入感很难掩盖包裹着玩家的surroundings，e.g人在地铁你问我代入感？
  • 第三人称
    • 1.更方便叙事的展开，更易摆脱时空对于叙事的限制，在景铺垫、多线叙事上都有体现。
    2.在价值判断的问题上玩家能有更超脱、客观的评判角度也能进行更自由的发挥（主角可以做恶，但引起的却是正向果，即玩家的反思，但第一人称下玩家是不会接受“自己”作恶剧情）。
